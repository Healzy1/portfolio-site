Projetos

CRUD com Firebase

Este projeto é um site criado seguindo meus gostos de anime, como se fosse um blog. Posteriormente o site foi atualizado para ser uma Aplicação de Página Única (SPA) e inclui um recurso CRUD para curiosidades de animes.

Funcionalidades:

O site possui um recurso CRUD (Criar, Ler, Atualizar, Deletar) para curiosidades de animes. Isso permite que qualquer usuário adicione novas curiosidades, leia, atualize ou delete curiosidades. Este recurso torna o site mais interativo e permite que os usuários compartilhem seus conhecimentos sobre animes.

Tecnologias Utilizadas: HTML, CSS, JavaScript e FireBase.

CRUD com Java e MySQL

Este projeto é uma aplicação de CRUD (Criar, Ler, Atualizar, Deletar) desenvolvida em JavaFX com integração ao banco de dados MySQL.

A aplicação permite salvar, consultar, atualizar e excluir dados de clientes, além de gerar dois relatórios específicos.

Funcionalidades:

Cadastro de Clientes: Permite a adição de novos clientes ao banco de dados.

Consulta de Clientes: Permite visualizar os dados dos clientes cadastrados.

Atualização de Clientes: Permite atualizar os dados dos clientes existentes.

Exclusão de Clientes: Permite deletar clientes do banco de dados.

Relatórios: Geração de dois tipos de relatórios:

- Lista de clientes do estado do Mato Grosso (MT).

- Lista de clientes que utilizam e-mails do Gmail.

Tecnologias Utilizadas: JavaFX, MySQL, JDBC, NetBeans e Scene Builder.
